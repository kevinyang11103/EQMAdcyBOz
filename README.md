# Python图书推荐系统 python496

## 项目概述

本项目是一个基于Python的在线图书推荐系统，通过协同过滤算法为用户提供个性化的图书推荐。

## 技术栈

- **开发语言及框架**：Python 3.8，Django 4
- **数据库**：MySQL 8
- **开发工具**：Navicat 数据库管理工具
- **前端技术**：HTML, JavaScript, jQuery, Bootstrap, Layer, WebUploader

## 功能模块

### 前台用户功能

- 注册
- 登录
- 注销
- 浏览图书
- 搜索图书
- 修改信息
- 修改密码
- 兴趣喜好标签
- 图书评分
- 图书收藏
- 图书评论
- 热点推荐
- 个性化推荐图书

### 后台管理员功能

- 用户管理
- 图书管理
- 图书类型管理
- 评分管理
- 收藏管理
- 评论管理
- 兴趣喜好标签管理
- 权限管理

## 个性化推荐功能

- 首页热点推荐：根据图书收藏数量进行降序推荐
- 个性化推荐：基于用户和项目的协同过滤推荐算法，依据评分数据提供推荐，若无推荐结果则依据喜好标签进行推荐

## 系统角色

- 前台用户：注册用户，可享受推荐系统各项服务
- 后台管理员：负责系统运营管理，维护图书数据及用户信息

## 运行环境

- 服务器：推荐使用Linux系统
- Python版本：3.8及以上
- 数据库：MySQL 8

## 目录结构

```
 project/
 ├── apps/              # 业务代码文件夹
 ├── static/           # 静态资源文件夹
 ├── templates/        # HTML模板文件夹
 ├── manage.py         # 管理脚本
 └── db/               # 数据库文件
```

## 部署步骤

1. 安装Python 3.8环境
2. 安装MySQL 8数据库
3. 克隆或下载项目代码
4. 导入数据库结构及数据
5. 配置应用环境
6. 运行`python manage.py runserver`
